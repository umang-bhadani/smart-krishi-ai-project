# Smart Krishi 1.0 🌱

**Empowering Farmers with Smart Soil Insights**

A modern, AI-powered web application that helps farmers make informed decisions about their crops by analyzing soil data and providing personalized farming recommendations.

## ✨ Features

### 🌾 Core Functionality
- **Soil Data Input Portal** - Comprehensive form for farmers to input soil parameters
- **AI-Powered Analysis** - Intelligent soil analysis and crop recommendations
- **Soil Knowledge Base** - Detailed information about different soil types
- **Responsive Design** - Optimized for mobile, tablet, and desktop devices

### 📊 Soil Parameters Supported
- Farmer information and location
- Soil type classification (Clay, Sandy, Loamy, Silty, Peaty, Chalky, Saline)
- pH level measurement
- Moisture content analysis
- Organic matter percentage
- Crop preference selection

### 🎯 Key Sections
- **Hero Section** - Engaging landing area with call-to-action
- **Soil Data Input** - Interactive form with validation
- **Soil Details** - Educational content about soil types
- **About** - Project mission and statistics
- **Contact** - Communication form and social links

## 🛠️ Tech Stack

- **Frontend Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Bolt Hosting
- **Code Quality**: ESLint + TypeScript ESLint

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/smart-krishi-1.0.git
   cd smart-krishi-1.0
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`

## 🎮 Usage

### For Farmers
1. **Enter Personal Information**: Provide your name and farming location
2. **Input Soil Data**: Fill in soil parameters like pH, moisture, and organic content
3. **Select Soil Type**: Choose from the dropdown menu of soil classifications
4. **Submit for Analysis**: Get AI-powered recommendations for your crops
5. **Explore Soil Knowledge**: Learn about different soil types and their properties

### For Developers
```bash
# Development
npm run dev          # Start development server
npm run build        # Build for production
npm run preview      # Preview production build
npm run lint         # Run ESLint
```
